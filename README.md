# Social_Media_WebApp
Built responsive social media UI with Next.js, implementing feed, profile, follow system, and notifications. Integrated backend APIs and handled dynamic data rendering with fallback states.
Built responsive UI using React.js & Next.js App Router with reusable component architecture.

Integrated REST APIs for posts, users, follow system, and notifications with proper loading states and fallbacks.

Implemented dynamic profile pages, followers/following lists, and real-time UI updates.

Designed scalable frontend structure with services layer, state handling, and role-based UI rendering.

Optimized performance using skeleton loaders, conditional rendering, and modular components.

Collaborated with backend developer for API contracts and data mapping.

Tech: React.js, Next.js, JavaScript, Tailwind CSS, REST APIs
